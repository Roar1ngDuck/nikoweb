BuggyNotes

An intentionally vulnerable web application developed using Django, designed to showcase common security flaws found in web applications. It includes features like user authentication, note creation, and backup functionalities, each implemented with known vulnerabilities such as SQL Injection, Cross-Site Scripting (XSS), and Insecure Deserialization.

WebChat

A discussion platform implemented using Flask, Docker, and PostgreSQL. It facilitates user interactions within various topic-based discussion areas and threads. WebChat emphasizes security with bcrypt for password hashing, Cloudflare Turnstile for CAPTCHA integration, and the zxcvbn library for password strength calculation.
